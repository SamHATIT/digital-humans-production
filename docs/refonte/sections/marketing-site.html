Site marketing & preview

Le site public Digital·Humans (preview en cours d'itération avant remplacement de la version en ligne) est un bundle React monolithique servi par Nginx. Format de fichier custom à 3 sections, modifications par mods appliqués par scripts Python.

Format du bundle

Le fichier /var/www/dh-preview/index.html (~14.5 MB) embarque tout : modules JSX, CSS inline, photos d'agents, textes bilingues. Trois sections <script type="..."> sont parsées au runtime :

SectionContenuVolume
__bundler/manifestJSON des ressources {uuid: {mime, compressed, data: base64}}41 entries (modules + photos)
__bundler/ext_resourcesMapping name → uuid (av0001, av0002, etc.)11 avatars + 1 carte projet
__bundler/templateJSON contenant le HTML complet de la page (CSS + modules JSX référencés par UUID)~1 MB

Modules JSX : compressés gzip + base64 ("compressed": true). Images : base64 brut ("compressed": false, ~600 KB-1.3 MB chacune en JPEG).

Workflow d'un mod

Chaque modification visuelle ou fonctionnelle est appliquée par un script Python docs/marketing-site/scripts/dh-mod{N}.py qui :

  1. Lit index.html → parse les 3 sections du bundle
  2. Modifie le module JSX cible (identifié par UUID stable, ex : b077057a pour avatars/CTA/Footer, 6641f2bf pour Benefits/HowItWorks)
  3. Re-compresse + ré-injecte dans __bundler/manifest
  4. Sérialise le template avec json.dumps(ensure_ascii=False) + escape </script> et </style> en \u002Fscript
  5. Sauvegarde en index.html.pre-mod{N}-{label} avant écriture (rollback en 1 commande)

Le site marketing communique avec le backend FastAPI via des routes publiques (no-auth, rate-limited) :

EndpointRôle

Service implémenté dans backend/app/services/sophie_concierge_service.py, model chat_log, route concierge_routes.py.

Pièges connus

  • ⚠️ Apostrophes ASCII dans les strings JSX cassent la string JS — toujours utiliser l'apostrophe typographique \u2019 pour les rôles FR.
  • ⚠️ Escape obligatoire de </script> et </style> dans le template JSON sérialisé (sinon parse HTML termine prématurément).
  • ⚠️ Cache navigateur — toujours Ctrl+Shift+R après modif pour voir le changement.
  • ⚠️ Backups index.html.pre-mod{N} non purgés automatiquement — gestion manuelle.
