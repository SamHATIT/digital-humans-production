Phase BUILD — Génération de code

Le BUILD transforme les specs SDS en code Salesforce déployable. Chaque agent BUILD utilise Sonnet 4.5 et reçoit les specs détaillées de Marcus comme contexte. Raj (Admin) intervient sur 3 phases, Jordan orchestre le déploiement de chaque phase via JordanDeployService, et Elena review chaque output avant deploy.
