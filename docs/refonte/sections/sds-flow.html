Révision Architecture (fix_gaps)

Quand Emma détecte un coverage < 85%, elle génère des gaps (what_is_missing + fix_instruction). Marcus reçoit ces gaps en mode fix_gaps (incrémental). Boucle auto-revision max 3 itérations.
